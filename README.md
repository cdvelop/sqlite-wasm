used as a sandbox to progressively isolate the SQLite engine and prepare a WebAssembly port
for [`github.com/tinywasm/sqlite`](https://github.com/tinywasm/sqlite).

## Status

🔧 **In development.** Do not use this module directly in production.

Once the work here is stable and tested, it will be merged back into `tinywasm/sqlite`.
